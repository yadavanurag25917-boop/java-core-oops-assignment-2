# SmartCampus Management System

## About
I built this project as part of my Java final assessment at GNC College.
It is a console-based system to manage students, courses and enrollments.

## Features
- Add Student
- Add Course (with fee validation)
- Enroll Student in Course
- View Students
- View Enrollments
- Multithreading for enrollment processing
- Custom Exception for invalid fee

## Files
- Student.java - Student class
- Course.java - Course class
- InvalidFeeException.java - Custom exception
- EnrollmentProcessor.java - Thread class
- CampusSystem.java - Main logic
- Main.java - Menu driven program

## How to Run
1. Open terminal
2. Type: javac *.java
3. Type: java Main
